# projet groupomania

## Prérequis d'installation
```
Installer  @vue/cli 

Installer le Systême de gestion de base de données mySql

installer la version 16.13.2 de node js 

installer la version 8.3.1 du gestionnaire de paquets npm

```

### Démarrage du projet 
```

Cloner le repository avec la commande git clone

Ouvrin un terminal à la racine du projet.

## Démarrage du frontend 
 - Se placer dans le fichier frontend2 avec cd frontend2
 - lancer la commande npm install
 - lancer la commande npm run serve pour demarrer l'application

 (En cas de succcès vous aurez un message de réussite)

## Démarrage du backend
-Se placer dans le dossier Backend avec cd Backend
- puis lancer la commande npm install
-puis lancer la commande npm run watch

À la création de la base de donnée

Remplacer l'username et le passsword par un utilisateur de votre SGBD.


 

 Ensuite placez-vous dans le répertoire **/frontend2** puis lancer la commande **npm run serve**

 Enfin placez-vous dans le répertoire **/Backend** et lancer la commande **npm run watch**

 ```

 > Bravo ! Le projet est lancée !
